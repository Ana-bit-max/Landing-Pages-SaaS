# 🚀 Taskflow - Landing Page SaaS (Em Construção)

Esta é uma landing page interativa para um SaaS fictício de gerenciamento de tarefas chamado **Taskflow**. O objetivo principal deste projeto é simular a experiência de um produto digital, oferecendo uma tabela de preços interativa e um fluxo completo de onboarding em várias etapas para o usuário testar antes do lançamento oficial.

> 🌐 **[Clique aqui para testar a simulação ao vivo no GitHub Pages](https://SEU-USUARIO.github.io/NOME-DO-REPOSITORIO/)** *(Substitua o link acima pelo link oficial que o GitHub Pages gerou para você)*

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído de forma leve e rápida, utilizando uma única estrutura de arquivo para facilitar o carregamento e a portabilidade:

* **HTML5:** Estrutura semântica da página.
* **Tailwind CSS (via CDN):** Estilização moderna, responsiva e em modo escuro (*dark mode*).
* **Vanilla JavaScript:** Toda a lógica de controle de estados do modal, validação do formulário e animação da simulação de carregamento do onboarding.
* **Font Awesome:** Ícones vetoriais modernos.

---

## 🎮 Funcionalidades Simuladas

1. **Visualização do Produto:** Um mockup do painel de controle (Dashboard) que convida o usuário a iniciar o teste.
2. **Seleção de Planos:** Cards com preços fictícios. Ao clicar em qualquer um deles, o sistema inicia o fluxo correspondente.
3. **Onboarding em 3 Etapas:**
   - **Etapa 1:** Coleta de dados (Nome, E-mail e Empresa).
   - **Etapa 2:** Personalização de objetivos (ajusta o painel com base nas escolhas).
   - **Etapa 3:** Tela de carregamento (*Loading*) com barra de progresso animada simulando a criação de um banco de dados.
4. **Revelação do Dashboard:** O usuário é recebido em um painel interativo personalizado com as informações coletadas na Etapa 1.
